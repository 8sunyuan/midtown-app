# Image Assets

Please add the following volleyball-themed images to this directory:

## Required Images

1. **volleyball-players-dark.png** - The black background image with green illustrated volleyball players
   - Used on the homepage hero section
   - Should show players in various volleyball poses (serving, jumping, etc.)

2. **volleyball-players-light.png** - The beige/cream background version with "MIDTOWN RUNS" text
   - Can be used for alternate sections or promotional materials

3. **scoreboard.png** - The detailed scoreboard graphic
   - Shows HOME/GUEST sections with digital display style
   - Can be used as a decorative element on standings pages

## Image Guidelines

- Save images in PNG format for transparency support
- Recommended dimensions: 1920x1080 or larger for hero images
- Optimize images for web (use tools like TinyPNG or ImageOptim)
- Maintain the green color scheme (#4ade80, #2d5f3f, #1a3a28)

## Current Theme Colors

- Primary Green: #4ade80
- Dark Green: #2d5f3f
- Darker Green: #1a3a28
- Accent Cream: #f5f1e8
- Black: #1a1a1a

These colors match the volleyball player illustrations provided.
